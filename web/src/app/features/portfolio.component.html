Project Glasswing — Computer Network Defense, TrenchWork.org

An open-source toolkit that turns public data into transparent, physics-grounded analysis — a procurement & R&D cost-benefit portfolio, interactive physics calculators, EW strategy, PLA/DoD doctrine, and a methodology page (physics, scoring math, CI parity, and fact corroboration). Every result traces to a textbook equation in the osint_cad Python package and is parity-checked in CI. Conceptual OSINT for education/study — not operational guidance and not a targeting system.
