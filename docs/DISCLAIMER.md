# Disclaimer & Scope

**Classification:** UNCLASSIFIED — derived entirely from open sources — for public release.

## What this project is

Defense-CAD is open analysis in the tradition of public defense scholarship (think CSIS,
RAND, FAS, or *Jane's*-style estimation). It uses **only publicly available information** —
photographs, manufacturer datasheets, published doctrine, academic papers, and physics
textbooks — together with first-principles physics to produce **bounded estimates** of system
parameters and engagement outcomes.

## What this project is not

- **Not classified.** No input to any model or reasoning chain comes from classified,
  proprietary, or export-controlled material. Where a real value is not public, this project
  derives an *estimate with an explicit uncertainty range* and labels it as such; it does not
  claim knowledge of the true (classified) figure.
- **Not ground truth.** Results are estimates, typically at 50–85% confidence with wide
  uncertainty bands. They can be — and sometimes are — wrong. That is why every chain shows its
  assumptions and lets you change them.
- **Not operational.** Nothing here is targeting data or validated for operational use. It is
  intended for education, research, and informed public discussion.

## On estimating sensitive parameters

Some parameters this project estimates (for example, a radar's element count or a datalink's
sidelobe level) are not publicly disclosed by their manufacturers. The estimates here are
produced by **derivation from public observables and physics**, with all work shown and all
uncertainty stated. A derived estimate that happens to land near a real value is a consequence
of physics constraining the design space — not disclosure of a protected fact.

If you believe any specific content is inaccurate, or crosses a line it should not, please open
an issue. Corrections and better-sourced inputs are welcome.
